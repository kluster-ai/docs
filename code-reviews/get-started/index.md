## Step 2: Pick your workflow

kluster.ai offers two review modes. Choose based on how you work:

| If you... | Use this mode |
|-----------|---------------|
| Code with AI assistants | [Agent Mode](/code-reviews/agent-mode/) |
| Write code directly in your editor | [Instant IDE Mode](/code-reviews/instant-ide-mode/) |
| Do both | Use both modes! |

Not sure? See [Pick your workflow](/code-reviews/get-started/pick-your-workflow/) for a detailed comparison.

## Step 3: Start reviewing

Once installed, Code Reviews works automatically:

- **Agent Mode**: Just start coding with your AI assistant. Reviews happen automatically.
- **Instant IDE Mode**: Right-click any code or press `Ctrl+Shift+K` to review.
